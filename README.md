# WEB102 Prework - Sea Monster Crowdfunding

Submitted by: Duban Correa

Sea Monster CrowfFunding is a website for the company Sea Monster Crowdfunding that displays information about the games they have funded.

Time spent: 12 hours spent in total

## Required Features

The following **required** functionality is completed:

* [ ] The introduction section explains the background of the company and how many games remain unfunded.
* [ ] The Stats section includes information about the total contributions and dollars raised as well as the top two most funded games.
* [ ] The Our Games section initially displays all games funded by Sea Monster Crowdfunding
* [ ] The Our Games section has three buttons that allow the user to display only unfunded games, only funded games, or all games.
* [x] Use filter to show only unfunded games.

* [x] Use filter to show only funded games.

* [x] Use toLocaleString to format currency with proper commas.

* [x] Implement a search/filter system using event listeners on buttons.

* [x] Use array destructuring to identify the top two most funded games.

* [x] Display a dynamic summary message using template literals and the ternary operator.

The following **optional** features could be implemented:

* [ ] Asearch bar to find games by name or a Sort by name botton
